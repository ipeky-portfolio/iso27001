# ISO 27001 Lead Auditor Portfolio - [Dein Name]

Willkommen auf meinem GitHub-Portfolio. Dieses Repository dient als Nachweis meiner fachlichen Kompetenz, methodischen Vorgehensweise und praktischen Erfahrung als zertifizierter **ISO/IEC 27001 Lead Auditor**.

Hier finden Sie exemplarische Dokumente, Vorlagen und methodische Ansätze, die ich in Audits nach ISO 27001:2022 anwende.

Das Repository ist strukturiert wie ein echtes Audit-Projekt:

1.  **Governance:** Unabhängigkeitserklärung & Kompetenznachweis.
2.  **Auditprogramm:** Strategische Jahresplanung & Risikobasierte Priorisierung.
3.  **Vorbereitung:** Auditpläne & Dokumentenprüfung.
4.  **Durchführung:** Stichprobenverfahren & Interview-Techniken.
5.  **Berichterstattung:** Executive Summaries & Abweichungsmanagement.
