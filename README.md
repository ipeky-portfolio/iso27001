# ISO 27001 Lead Auditor Portfolio

This repository demonstrates a full-scope Information Security Management System (ISMS) audit aligned with ISO/IEC 27001:2022 and ISO 19011 auditing principles.

## Repository Structure

- 00_Audit_Governance → Audit charter, independence, competence
- 01_Regulatory_Framework → ISO overview & certification logic
- 02_Audit_Programme → Risk-based annual planning
- 03_Audit_Execution → Planning, sampling, evidence handling
- 04_Control_Testing_Modules → Clause & Annex A control testing
- 05_Findings_and_Reporting → Formal audit report & findings
- 06_Corrective_Actions → CAPA & effectiveness validation

## Demonstrated Capabilities

- Risk-based audit planning
- Evidence-based control testing
- Nonconformity classification (Major/Minor/OFI)
- Management-level reporting
- Certification readiness assessment
- Traceability: Risk → Control → Evidence → Finding → CAPA

## Disclaimer

This repository contains a simulated audit case for professional portfolio purposes.
